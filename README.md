# Contacts Manager ( CRUD ) 
It is a Contacts Manager constructed with Java,and PostgreSQL, using the Javafx framework . 

- Add new contact .
- Edit existing  contact  (Edit/Delete) .
- Show all contacts in a table .
- Moves between the scenes .
- Simple UI .
